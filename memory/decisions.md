# Decisions Log

> **MANDATORY:** Append to this file whenever a significant decision is made during Planning or Implementation.

## How to Use

After making a decision, append a new entry:

```markdown
## [YYYY-MM-DD]: [Task/Feature Name]

### Decision: [What was decided]

**Context:** [Why this decision was needed]

**Options Considered:**
1. [Option A] - [pros/cons, why rejected]
2. [Option B] - [pros/cons, why chosen]

**Chosen:** Option [X]

**Reasoning:**
- [Key reason 1]
- [Key reason 2]

**Trade-offs Accepted:**
- [Trade-off 1]

**Evidence:**
- Similar pattern: `file:line`
- Constraint: `constraints.md:line`
```
