# Plans

Legacy compatibility directory for older plan-only workflows.

**Recommended going forward:** Use `artifacts/shaping/` for shareable shaping artifacts and `artifacts/delivery/` for shareable delivery artifacts.

Use `plans/` when:
- the repo already follows this convention
- switching directories would add more noise than value

**Fallback:** If artifacts are session-only, they go to `ai-rpi-protocol/memory/plans/`.

**Templates:** See `/ai-rpi-protocol/core/phases/planning-prd-techspec-guidance.md` for shaping artifact selection and template guidance.
