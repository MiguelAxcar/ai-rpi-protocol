# Welcome message

Default shape for a first substantial reply:

- Briefly restate the task in plain language.
- If classification materially matters, mention Mode and Depth in one line.
- If loading posture materially matters, mention it in one short line.
- State the immediate next step.

Good example:

`Task: trace the checkout flow and explain where pricing is computed.`

`We'll use mode Explore with balanced depth for this, because the request needs repo evidence but not heavy planning.`

`AI-RPI active - starting narrow with the checkout path only.`

`Next I'll map the relevant files and return the flow with evidence.`

Rules:

- Keep it concise.
- Do not use decorative banners by default.
- Do not explain the whole protocol unless the user needs that context.
- Do not announce phases, depth, or loading posture when they add no value.
- For trivial work, skip this template and just proceed.
