# Restate Other Request

**When to use:** User asks for code review, refactoring, explanation, optimization, testing, documentation, cleanup, migration, or design work. Use BEFORE Phase R on non-trivial requests.

## Format

**Request:** [One sentence describing what the user wants]

**Type:** [Code review / Refactoring / Explanation / Optimization / Testing / Documentation / Cleanup / Migration / Design]

**Focus areas:**
- [Focus area 1]
- [Focus area 2]

**Excluded:**
- [Excluded area 1]

**Expected outcome:**
- [Outcome 1]
- [Outcome 2]

**Considerations:**
- [Consideration 1] — impact: [why it matters]
- [Consideration 2] — impact: [why it matters]

**Questions:**
1. [Question about scope/preferences]
2. [Question about constraints]

**Next:** I'll research [what], plan [what], then execute [what].

Does this match what you want? [yes/no/adjustments]
